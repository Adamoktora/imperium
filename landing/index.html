The Problem

Crypto portfolios are fragmented across Ethereum, Base, Polygon, Arbitrum with no unified view. You can't detect rug pulls, can't see total allocation, and rebalancing means hours of manual swaps across different UIs.

The Solution

Imperium gives you one command center for everything: portfolio tracking, risk scoring, smart money discovery, and policy-gated rebalancing - all powered by MoonPay CLI via real MCP protocol.

Architecture

User (CLI / Interactive REPL) | v index.ts (command router) | +---> core/portfolio.ts ---+ +---> core/risk.ts ---+ +--> MoonPay CLI (MCP) +---> core/rebalance.ts ---+-- policy gate ---+ +---> core/discovery.ts ---+ (wallet/policy) +--> OWS Wallet (sign) | ~/.imperium/config.json (persistent state)

Core Modules

Portfolio

Holdings across chains, allocation %, PnL tracking, transaction history, fiat onramp

Risk Scanner

Token safety check: holder concentration, liquidity analysis, community warnings. Scores 0-100 with HOLD/WATCH/SELL

Rebalance

Set target allocation, preview drift, auto-generate swap/bridge actions. Policy-gated execution with OWS signing

Discovery

Trending tokens, smart money wallets (top PnL), token search. Real data from MoonPay

Policy Engine

Daily spending limits, per-tx caps, token blocklists, slippage guards. OWS-signed attestation

OWS Wallet

Real HD wallet creation via OpenWallet Standard. 7 chains: EVM, Solana, Bitcoin, Cosmos, Tron, TON, Filecoin

Tech Stack

MCP Client

@modelcontextprotocol/sdk - official SDK, real stdio transport to MoonPay CLI

Wallet

@open-wallet-standard/core - real HD wallet, NAPI-RS native binary, 7 chains

Testing

Vitest - 31 tests across 7 files (unit, integration, policy, rebalance)

Language

TypeScript (strict mode) with ESM modules

CLI

Commander.js + chalk + cli-table3 for rich terminal output

Data

14 fixture files captured from real MoonPay API responses

Quick Start

# Clone and install $ git clone https://github.com/jordi-stack/imperium.git $ cd imperium && npm install # Demo mode (no funds needed) $ npx tsx src/index.ts --demo portfolio $ npx tsx src/index.ts --demo risk scan $ npx tsx src/index.ts --demo discover trending base $ npx tsx src/index.ts --demo interactive # Real mode (requires MoonPay CLI login) $ npm install -g @moonpay/cli $ moonpay login --email you@example.com $ npx tsx src/index.ts discover trending base
